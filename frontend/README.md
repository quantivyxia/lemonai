# Frontend InsightHub

Documentação principal:

- veja o `README.md` na raiz do repositório

Comandos locais:

```bash
npm install
npm run dev
npm run lint
npm run build
```

Variáveis de ambiente:

- `VITE_API_BASE_URL`
- `VITE_API_TIMEOUT_MS`
- `VITE_LOG_LEVEL`

Observação:

- este frontend usa Vite; a saída de build é `dist/`
